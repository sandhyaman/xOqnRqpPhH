# 学生成绩管理系统 python51

## 项目概述

学生成绩管理系统是一款基于Python语言和MySQL数据库的教育信息化产品，旨在帮助学校管理学生成绩，提高工作效率。

## 技术栈

- **后端**: Python
- **Web框架**: Flask
- **数据库**: MySQL

## 功能模块

- 学生信息管理
- 课程信息管理
- 成绩录入与查询
- 成绩统计分析

## 系统角色

- 系统管理员
- 教师
- 学生

## 运行环境

- **操作系统**: Linux/Windows/MacOS
- **Python版本**: 3.6 或以上
- **数据库**: MySQL 5.7 或以上

## 部署步骤

1. 准备Python运行环境
2. 安装依赖库：Flask、MySQL-Python等
3. 配置MySQL数据库
4. 运行部署脚本
5. 启动服务
